# Viited

See leht koondab peamised [`mkdocs-material`](https://squidfunk.github.io/mkdocs-material/)-i referentsid, mida saab selle dokumentatsiooni edasiarendamisel kasutada.

## Põhireferentsid

| Teema | Milleks kasutada | Link |
| --- | --- | --- |
| Üldine referents | Kõik saadaolevad sisuelemendid ühel lehel | [Reference](https://squidfunk.github.io/mkdocs-material/reference/) |
| Märkusplokid | Info, hoiatused, soovitused | [Admonitions](https://squidfunk.github.io/mkdocs-material/reference/admonitions/) |
| Vahelehed | Sisu jaotamine plokkidesse | [Content tabs](https://squidfunk.github.io/mkdocs-material/reference/content-tabs/) |
| Jalused | Lisaselgitused teksti sees | [Footnotes](https://squidfunk.github.io/mkdocs-material/reference/footnotes/) |
| Pildid | Illustratsioonide paigutamine | [Images](https://squidfunk.github.io/mkdocs-material/reference/images/) |
| Tabelid | Ajakavade ja andmete kuvamine | [Data tables](https://squidfunk.github.io/mkdocs-material/reference/data-tables/) |

## Kuidas neid viiteid kasutada

1. Vali sobiv referents vastavalt sellele, kas soovid lisada märkust, pilti, tabelit või vahelehti.
2. Kopeeri näitesüntaks dokumentatsiooni Markdown-faili.
3. Kohanda tekstid ja lingid selle projekti sisule vastavaks.

!!! example "Praktiline soovitus"
    Kui soovid sammud või variandid kõrvuti kuvada, kasuta kõigepealt [Content tabs](https://squidfunk.github.io/mkdocs-material/reference/content-tabs/) referentsi. Kui soovid rõhutada tähtsat infot, kasuta [Admonitions](https://squidfunk.github.io/mkdocs-material/reference/admonitions/) plokke.
